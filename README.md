## Features 
- Read QR Codes with your camera
- Open sites and maps stored in bookmarks 

## What did i use in this app? 

- Launch the camera and read a QR code
- Store it in SQLite
- CRUD to SQLite - Internal storage
- use of maps
- Google Maps
- Markers
- Open URLs
- Provider as state manager
